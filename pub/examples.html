The Problem

We do not want web elements to overlap each other during animations.

Example 1

Using the Seui2 library, you can avoid the element like so.

 Seui('#example1 svg', 'ex1-animate', [
    {
      avoid: '#ex1-avoid',
      start: {x: 873, y: 100},
      end: {x: 33, y: 100},
    },
  ]);

Go here to see an explanation and get started.

Click the smaller picture to animate it.

Interactive Example

In the example below, you can set the start and end position for the element to animate. Where the coordinates specify the top-left corner of the object. This is the same for the position of the element to avoid. You can also change the width and height of both. The library will take this into account and generate the new path. Click the blue rectangle to animate it.
